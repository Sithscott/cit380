Challenge Path details

Overview

The main focus of the Challenge path is to allow students to experiment deeply with the idea of "Acting and not being acted upon". You are given great flexibility with this path to determine everything about your project including:

  • Project management methodology
  • Deliverables
  • Milestones
  • Assessment method (how your project should be graded)

The assessment piece in particular should be negotiated and ultimately approved by the instructor. A challenge path project should be a group project.

Additional information

A Project Management Plan (PMP) is a living document. Once you have completed yours make sure to regularly update it as necessary while you execute the project.
